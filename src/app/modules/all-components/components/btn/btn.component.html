You can also combine the buttons with the ripple directive, to create a ripple effect when the button is clicked.

See more about ripples here

Buttons also have the option to be elevated. When a button is elevated, it will cast a shadow. The density of the shadow will change when hovered and clicked.

FAB buttons are completely rounded buttons, which generally represent the main action of a page.


An FAB has a fixed width and height, so their actions are best representend by a single icon without any text.
