Usage

Buttons are used to perform actions through the application.


Behaviour

The buttons behave like a normal HTML <button> element, and accepts all of angular bindings.


Styling

The Productive custom button stylings are achieved by simple css classes. You just need to add the classes listed in the references tab to a regular <button> element to achieve the styling.
