Each of the 12 columns are divided by 3 types of screen: Small, Medium and Large. So you can set a specific amount of columns for an element to occupy on a certain screen size.

For reference, here are the screen width values in pixels for each type:
  • Small: 600px and bellow;
  • Medium: 1000px and bellow;
  • Large: 1001px and above;
