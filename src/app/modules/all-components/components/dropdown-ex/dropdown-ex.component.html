Usage

A dropdown menu is a hidden menu that contains actions triggers, and is separated from the normal page flow.


Examples

Basic Menu

Menu buttons can also have icons, to better illustrate the action it will perform.


Additionally, the menu's width will adjust automatically depending of the content's size.


Basic Menu with Icon Button

Menus automatically adjust their positioning, however you can set a fixed position for the menu to trigger.


Basic Menus with custom positioning

There's also an option to those that do not want the menu to close when a button inside it is clicked, and also an option to open the menu by just hovering over the trigger element.
