Usage


The Ripple Effect is used to represent when the user has clicked something, sending a visual feedback to the user.

Here's an interactive example of the Ripple Effect:
Click Me!

You can also set up an element to be a trigger of a ripple on another element.

When doing so, the ripple will always be centered, however every other setting will work as intended.

Here's an example: 
            
              <div class="ripple-example elevation-p8" pRipple [pRippleTriggerFor]="rippleTrigger"></div>
              <div class="ripple-example elevation-p8" #rippleTrigger></div>
